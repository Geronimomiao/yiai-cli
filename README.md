# yai-cli
init project from cli tool

## Insall

``` bash
npm install -g yai-cli
```

## Init Project
### Init manage-system project

``` bash
yai manage-system-template my-project
```

### Inint electron-app project

``` bash
yai electron-vue-template my-project
```
