# yiai-cli
init project from cli tool

## Insall

``` bash
npm install -g yiai-cli
```

## Init Project
### Init manage-system project

``` bash
yiai manage-system-template my-project
```

### Inint electron-app project

``` bash
yiai electron-vue-template my-project
```
